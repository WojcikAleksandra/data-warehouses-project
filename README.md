# Power Outages Data Warehouse Project

This project implements a Business Intelligence solution for analyzing power outages in the United States and their relationship with weather conditions, 
extreme weather events, and socio-economic factors. The solution includes an ETL process, a SQL Server data warehouse, an OLAP cube built with SQL Server 
Analysis Services (SSAS), and Power BI dashboards.

This is a team project completed as part of the *Data Warehouses and Business Intelligence Systems* academic course.

## Data Sources

The system integrates data from multiple sources, including:
- power outages data - https://figshare.com/articles/dataset/The_Environment_for_Analysis_of_Geo-Located_Energy_Information_s_Recorded_Electricity_Outages_2014-2022/24237376, 
- weather data - https://www.aaronsmithagecon.com/download-us-weather-data, 
- NOAA storm events data - https://www.ncei.noaa.gov/pub/data/swdi/stormevents/csvfiles/, 
- U.S. Census socio-economic data - https://www.census.gov/data/developers/data-sets.html, 
- county geographic data - https://github.com/davidingold/county_centroids.
